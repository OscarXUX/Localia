# Localia: Ola México 2026 🇲🇽

**Localia** es una plataforma móvil diseñada para potenciar la visibilidad de las PyMEs locales en Guanajuato durante la Copa del Mundo 2026, impulsando la inclusión financiera a través de **Coppel Pay**.

## 🚀 Tecnologías
- **Framework:** Flutter (Dart)
- **State Management:** Provider (Clean Architecture)
- **UI/UX:** Diseño basado en principios de Apple (Bordes redondeados, Glassmorphism).
- **IA:** Asistente inteligente simulado para traducción y recomendaciones turísticas.

## 📂 Arquitectura del Proyecto
Aplicamos una estructura modular para facilitar la escalabilidad:
- `models/`: Definición de entidades (Business, Event).
- `providers/`: Lógica de negocio y gestión de estados globales.
- `ui/`: Pantallas e interfaces de usuario divididas por roles (Turista y Admin).
- `widgets/`: Componentes reutilizables y atómicos.

## 🎯 Funcionalidades Clave
1. **Perfil Dual:** Acceso diferenciado para turistas y dueños de negocios.
2. **Coppel Pay Integration:** Simulación de pagos rápidos para PyMEs.
3. **Mundial Mode:** Visualización de partidos en tiempo real cerca de los locales comerciales.
4. **Filtros Inteligentes:** Búsqueda reactiva por nombre y categoría.
